# Automatic Speech Recognition (ASR) System

End-to-end Automatic Speech Recognition system built from scratch using PyTorch.

## Features

- Log-Mel spectrogram preprocessing
- CNN + BiGRU acoustic model
- CTC loss with greedy and beam search decoding
- Character Error Rate (CER) & Word Error Rate (WER)
- Gradio web demo
- Whisper comparison (transfer learning)

## Tech Stack

- Python
- PyTorch
- Librosa
- Gradio
- Hugging Face Transformers
